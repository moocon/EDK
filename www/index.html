Welcome to the EDK PROJECT, TianoCore's first and most prominent project. The EDK is the open-source component of the "Framework", Intel's implementation of the EFI Specification, which was developed under the project codenamed "Tiano". Refer to the EDK section of the TianoCore FAQs for a full explanation of how the EDK relates to the EFI effort.

NEW!Modules Updated

As a result of ongoing work to modularize the codebase, we now have an updated list of modules for the EDK, and owners for each module. Check out EDK Module Structure Explained for more information.

The Purpose of the EDK

The EDK serves as the Open Source container for the Framework's Foundation code and sample drivers. The EDK is also a development kit for developing, debugging, and testing EFI and Framework drivers, EFI Option ROMs, and EFI Applications for use in the Framework environment.

Who Should Join this Project

As an anonymous user you can download and build the EDK. As a TianoCore registered user you can view the mailing list archives, discussion forums and defects. However its only through being a project member that you can truly be a part of the EDK's future. Here are a few types of companies may want to consider becoming a part of the EDK development team:

  • Independent Hardware Vendors (IHVs)
  • Independent BIOS Vendors (IBVs)
  • Original Equiment Manufacturers (OEMs)
  • Operating System Developers
  • Open BIOS Engineer

Working with the EDK

The EDK project is sponsored by Intel and released under the BSD license. To download snapshots of the current development or official releases of the EDK go the directory of EDK source code snapshots and choose the version (Official Release or Development Version) you are interested in and simply click on it to start the download process. Join this project if you want to help maintain and enhance the EDK (see the next section for guidance).

Guidance on buiding the EDK and using it to develop and test drivers can be found in the EDK Getting Started Guide (pdf) and the Framework Specification on the Intel website.

Joining the EDK Project

To join the EDK project simply click on the "Request Project Membership/Role" link above and choose "Observer" as a requested role (the default role for all new project members). Upon receipt of your request, the EDK Maintainer will approve your project membership. Once in the project familiarize yourself with the project tools such as:

  • Mailing Lists: Here you will find mailing lists available exclusively for the EDK project members.
  • Discussion Forums: Discuss topics related directly to the EDK project (the TianoCore Community Project has other interesting forums as well)
  • Documents & Files: This section of the project houses important EDK-specific documents, such as specs, code snapshots, how-to's, etc.
  • EDK Source Code: This is the CVS-based source code repository and access instructions.
  • Project Tracker (issue management system): This is an issue management database exclusively for EDK-related bugs, enhancements, tasks, etc.
  • EDK-specific FAQs

The most common starting point for new members is to download the latest EDK Developer Snapshot and become familiar with the current Open and Active EDK Defects . To see how one goes about becoming part of the team, gaining a project role, authority and permissions, please take a look at our "EDK Development Organization" page. To understand our defect resolution process (which is actually pretty structured for an Open Source project), see the "Issue Tracking Explained".

If you want to be a part of the greater TianoCore community you are highly encouraged to join the TianoCore Community Project as well.
